# FastAPI-ToDoAPI

## Setup Develop Environment

1. `python3 -m venv .venv`
2. `source .venv/bin/activate`
3. `pip install -r requirements.txt`

## Build

`pyinstaller -F main.py --clean`

## Use

### Start Server

`dist/main`

### Read Swagger

Access http://0.0.0.0:8000/docs

### Connect API

Domain: http://0.0.0.0:8000/
